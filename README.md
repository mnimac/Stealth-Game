# Stealth-game
2.5D video game
 
------------------------------------------------------------------------------------------------------------------------------------------
## How to play?

Reach the blue finish tile without being seen by the Guards and their Spotlight.
> Use `WASD` or `arrow keys` for movement.

>To exit the game press `Esc`.
